# Example Workflows

Worked examples showing the iterative AI branding process from initial input through refinement to final output.

## Example A: Color Palette for a Tech Startup

**User input:** "Modern tech startup in the developer tools space. Should feel professional but not corporate. Target audience is developers aged 25-40."

### Iteration 1 — Initial Suggestions

**AI suggests 3 palette directions:**

1. **Electric Focus** — Deep charcoal `#1A1A2E` + electric blue `#0EA5E9` + warm white `#FAFAF9`
2. **Forest Code** — Dark green `#064E3B` + lime accent `#84CC16` + slate `#334155`
3. **Sunset Terminal** — Near-black `#0C0A09` + amber `#F59E0B` + stone `#78716C`

**User feedback:** "I like the energy of option 1 but the blue is too generic — every dev tool uses blue. Option 3's amber is interesting. Can you combine the dark base of 1 with warmer accents?"

### Iteration 2 — Refined Based on Feedback

**Locked:** Dark base direction from option 1
**Rejected:** Generic blue, green direction
**Adjusting:** Warm accent colors inspired by option 3

**AI suggests refined palette:**
- Primary: `#16161A` (rich near-black)
- Secondary: `#242629` (elevated dark surface)
- Accent: `#E8A838` (warm amber-gold)
- Highlight: `#FF6B35` (energetic orange for CTAs)
- Neutral: `#94A1B2` (cool gray for text)

**User feedback:** "Yes! The amber-gold is perfect. The orange CTA might be too aggressive — can you soften it? Also need a success/error color."

### Iteration 3 — Final Refinement

**Locked:** Primary, secondary, accent (amber-gold)
**Adjusting:** CTA color, adding semantic colors

**Final approved palette:**
- Primary: `#16161A`
- Secondary: `#242629`
- Accent: `#E8A838`
- CTA: `#D4784B` (softened terracotta)
- Success: `#4ADE80`
- Error: `#F87171`
- Neutral: `#94A1B2`

**Result:** Converged in 3 iterations. The palette avoids generic dev-tool blue while maintaining a professional, modern feel.

## Example B: Typography Pairing for a Health & Wellness Brand

**User input:** "Wellness app for meditation and mindfulness. Calm, approachable, slightly premium. Users are 30-55, mix of genders."

### Iteration 1 — Initial Suggestions

**AI suggests 2 pairings:**

1. **Serene Editorial** — Heading: Playfair Display (serif, elegant) / Body: Source Sans 3 (clean sans)
2. **Soft Modern** — Heading: DM Sans (rounded geometric) / Body: Inter (neutral, readable)

**User feedback:** "Playfair feels too formal, like a luxury hotel, not a wellness app. DM Sans is closer but too plain. Something with more warmth?"

### Iteration 2 — Refined

**Rejected:** Formal serifs, overly neutral sans
**Direction:** Warm, rounded, approachable but not childish

**AI suggests:**
- Heading: **Nunito** (rounded terminals, friendly weight range, 700 for headings)
- Body: **Karla** (humanist sans, warm curves, excellent readability at 400/16px)

Size scale: 14 / 16 / 20 / 28 / 36 / 48px with 1.5 line-height for body, 1.2 for headings.

**User feedback:** "Nunito + Karla feels right. Can we see how it looks with slightly tighter heading line-height and a lighter weight for large headings?"

### Iteration 3 — Final Adjustment

**Locked:** Nunito + Karla pairing, body size/weight
**Adjusting:** Heading weight and line-height

**Final approved typography:**
- Heading: Nunito, 600 weight (was 700), line-height 1.15 (was 1.2)
- Body: Karla, 400 weight, line-height 1.5
- Small: Karla, 400, 14px
- Caption: Karla, 500, 12px, uppercase tracking 0.05em

**Result:** Converged in 3 iterations. Warm and approachable without being childish.

## Example C: Full Brand Kit (End-to-End)

**User input:** "E-commerce platform for handmade artisan goods. Think Etsy but more curated and premium. Target: buyers who value craftsmanship, ages 28-50."

### Iteration 1 — Full Brand Elements

**Colors:**
- Cream base `#FAF7F2` + forest green `#2D5016` + clay `#C4724E` + charcoal `#2C2C2C`

**Typography:**
- Heading: Fraunces (variable serif, quirky optical sizing)
- Body: Work Sans (friendly geometric)

**Imagery:** Hand-drawn textures, warm photography with natural light, organic shapes

**User feedback:** "Love the earthy direction. The green feels too dark and heavy — it's competing with the clay. Fraunces is interesting but might be too quirky for a premium feel. Keep the cream and clay."

### Iteration 2 — Palette + Typography Refined

**Locked:** Cream `#FAF7F2`, clay `#C4724E`, charcoal `#2C2C2C`, imagery direction
**Adjusting:** Green → lighter, heading font → less quirky

**Refined:**
- Green adjusted: `#4A7C3F` (lighter, more botanical)
- Heading changed to: **Cormorant Garamond** (elegant serif, premium without being stiff)
- Body kept: Work Sans

**User feedback:** "Much better balance. The green works now. Cormorant Garamond is beautiful. Can we add a warm gold accent for premium elements like badges and featured items?"

### Iteration 3 — Add Accent + Final Lock

**Locked:** Full palette (cream, green, clay, charcoal), typography, imagery
**Adding:** Gold accent

**Final brand kit:**
- Cream: `#FAF7F2` (backgrounds)
- Forest: `#4A7C3F` (primary actions, navigation)
- Clay: `#C4724E` (secondary actions, highlights)
- Charcoal: `#2C2C2C` (text, structure)
- Gold: `#B8860B` (badges, featured items, premium indicators)
- Heading: Cormorant Garamond, 500/600 weight
- Body: Work Sans, 400 weight
- Imagery: Warm natural photography, hand-drawn textures, organic rounded shapes

**Result:** Converged in 3 iterations. Premium artisan aesthetic with a warm, curated feel.
